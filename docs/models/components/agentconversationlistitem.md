# AgentConversationListItem

Conversation summary returned by `GET /agents/{agentKey}/conversations`.
The handler excludes `messages` and `__v` from both result sets. Rows
in `sharedWithMeConversations` also omit `sharedWith` because the
secondary query explicitly deselects that field before serialization.
