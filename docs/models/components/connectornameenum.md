# ConnectorNameEnum

`Connectors` enum (`backend/python/app/config/constants/arangodb.py`);
records store the enum value (e.g. Google Drive is `DRIVE`,
SharePoint Online is `SHAREPOINT ONLINE`), not the enum member name.
